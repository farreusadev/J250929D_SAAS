# J250929D_SAAS

Django SAAS Project

## Description
This is a Django-based SAAS application project.

## Installation
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Configuration
Configure your database and other settings in `config/settings.py`

## Project Structure
- `config/` - Django project configuration
- `manage.py` - Django management script
- `templates/` - HTML templates directory
